Web Server

What is a web server?

A web server is a computer system that processes requests via HTTP, the basic network protocol used to distribute information on the World Wide Web.

What does a web server?

The primary function of a web server is to store, process and deliver web pages to clients.

How can content be stored on a web server?

Content can be stored static, as a real file on the server, or dynamic.

How is content in webpages described?

Web pages delivered are most frequently HTML documents, which may include images, style sheets and scripts in addition to text content.

Web Client

What is a web client?

A web client (commonly referred to as a browser) is a software application for retrieving, presenting and traversing information resources on the World Wide Web.

What does a web client?

Although browsers are primarily intended to use the World Wide Web, they can also be used to access information provided by web servers in private networks or files in file systems.

Declaration for background image

The background-image property specifies an image to use as the background of an element. By default, the image is repeated so it covers the entire element.

Declaration for text alignment (left, right, center, justify)

The text-align property is used to set the horizontal alignment of a text. A text can be left or right aligned, centered, or justified.

Declaration for text decoration (overline, underline, line-through)

The text-decoration property is used to set or remove decorations from text.

Declaration for text transformation (uppercase, lowercase, capitalize)

The text-transform property is used to specify uppercase and lowercase letters in a text.

Identify the difference between serif and sans-serif fonts

Serif fonts have small lines at the ends on some characters
"Sans" means without - these fonts do not have the lines at the ends of characters

Declarations for font families

The font-family property should hold several font names as a "fallback" system. If the browser does not support the first font, it tries the next font, and so on.

Declarations for font style normal and italic

normal - The text is shown normally
italic - The text is shown in italics

Declarations for font sizes

The font-size property sets the size of the text.

Declarations for font weights

The font-weight property specifies the weight of a font.
